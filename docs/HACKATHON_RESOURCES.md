# Hackathon Resources

## Main Track

AgentSea targets the Agentic Commerce track.

The core demo flow is:

1. Hamburg Cargo agent requests maritime intelligence.
2. AgentSea returns x402 payment requirement.
3. Agent pays on Algorand.
4. AgentSea verifies payment.
5. AgentSea returns structured maritime intelligence.
6. Agent uses the intelligence to make a business decision.
